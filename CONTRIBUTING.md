# Contributing

Bug reports, feature suggestions and usage questions are all very welcome —
please open an issue. This is also the way to reach the maintainer for
support.

Pull requests are **not accepted at this time**. This project is dual-licensed
(AGPL-3.0-or-later plus a separate commercial license; see
[LICENSE](LICENSE) and the "License / Commercial use" section of the
README), and accepting external code contributions would require a
Contributor License Agreement (copyright assignment) to keep that dual
licensing possible. Until that process exists, please open an issue to
discuss a change instead of sending a PR.
